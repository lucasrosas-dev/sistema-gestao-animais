# Sistema de Gestão de Animais — versão 1.0.0

Primeira versão pública do sistema.

## Conteúdo

- gestão completa de animais e produção;
- custos, receitas e resultados financeiros;
- relatórios e exportações;
- usuários, perfis e auditoria;
- proteção de formulários, sessões e login;
- suporte local a SQLite e online a PostgreSQL;
- publicação no Render com banco Neon;
- health checks e migrations Alembic.

## Ambiente

- Python 3.14.3;
- FastAPI 0.139.2;
- SQLAlchemy 2.0.51;
- Alembic 1.18.5;
- Uvicorn 0.51.0.

O pacote não contém credenciais nem dados de exemplo.
